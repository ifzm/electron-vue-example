# electron-vue

[vue-cli-plugin-electron-builder](https://nklayman.github.io/vue-cli-plugin-electron-builder)

## Project setup

```sh
yarn install
```

### Compiles and hot-reloads for development

```sh
# web
yarn serve
# client
yarn electron:serve
```

### Compiles and minifies for production

```sh
# web
yarn build
# client
yarn electron:build
```

### Lints and fixes files

```sh
yarn lint
```

### Publish

```sh
yarn release
```
